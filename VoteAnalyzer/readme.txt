Приложение считывает данные из XML файла о визитах на избирательный участок.
Фамилия, имя, дата рождения, номер участка и время голосования.
Эти данные переносятся в базу данных, после чего печатается список проголосовавших
более одного раза, а также время работы каждого участка.

В приложении используются:
MySQL
