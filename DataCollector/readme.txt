Приложение подключается к сайту и записывает в формате JSON схему линий московского метро.
Далее в папке с данными ищест JSON и CSV файлы с информацией о станциях, 
они содержат неполную информацию, например только название и глубину залегания. 
После чего записывается JSON файл с полной информацией обо 
всех станциях.
