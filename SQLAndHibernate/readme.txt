Приложение создаёт таблицы в базе данных с использованием Hibernate.
После создания таблиц необходимо импортировать данный из файла в базу данных
(находится в папке resources).
После этого необходимо повторно запустить приложение, заполнится таблица 
уникальных сочетаний "студент - изучаемый курс".

В приложении используются:
MySQL
Hibernate
